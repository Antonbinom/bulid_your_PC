# Custom PC

Pet-проект. Промо-страница.

## О приложении

- Адаптивная и валидная верстка
- БЭМ
- Кроссбраузернность
- Плавный скролл
- Таймер с обратным отсчетом до конца акции
- Валидация формы
- Отправка формы на сервер

## Стэк технологиий:

- Отправка формы через Fetch
- Sass
- Инкапсуляция JS
